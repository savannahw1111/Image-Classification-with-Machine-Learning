# Image-Classification-with-Machine-Learning-
Stack: R • caret • randomForest • kernlab • pROC • ggplot2

Built and evaluated machine learning models to classify cell image segmentation quality using 118 imaging features. Implemented stratified train/test splitting and cross-validation to ensure robust model performance.

Trained a Random Forest model with feature selection (Top 10 variables) to improve efficiency and accuracy, achieving ~85% test accuracy. Also developed a Support Vector Machine (RBF kernel) optimized using ROC metrics.

Generated confusion matrix visualizations and compared model performance to identify the strongest classification approach.
